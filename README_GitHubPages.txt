GitHub Pages Publish — KJ’s Mobile Dog Nail Services

Option A — Web UI (no command line):
1) Go to https://github.com and sign in.
2) Click the + (top-right) → "New repository". Name it: kjs-mobile-dog-nail-services
3) Leave it Public. Click "Create repository".
4) Click "Upload files" and drag the two files from the ZIP into GitHub:
   - index.html
   - styles.css
   (You can also upload favicon.ico and README_GitHubPages.txt, optional.)
5) Click "Commit changes".
6) Open the repo’s Settings → Pages (left sidebar).
7) Under "Build and deployment", choose:
   - Source: "Deploy from a branch"
   - Branch: "main" (or "master") / folder: "/ (root)"
8) Click Save. After 1–2 minutes, GitHub shows your live site URL:
   https://<YOUR-USERNAME>.github.io/kjs-mobile-dog-nail-services/

Option B — Command line (macOS/Linux/Windows with Git):
git init
git add index.html styles.css favicon.ico README_GitHubPages.txt
git commit -m "Initial site"
git branch -M main
git remote add origin https://github.com/<YOUR-USERNAME>/kjs-mobile-dog-nail-services.git
git push -u origin main
# Then enable Pages in Settings → Pages as above.
